This plugin uses the read more link and opens the post in modal window.<br />
Install it and activate it to see it in action.<br />
Go to Settings->Modal Read More and check the options.

== Installation ==

1. Upload the folder "modal-readmore" to the "/wp-content/plugins/" directory or install it through WordPress directly.
2. Activate the "Modal Read More" plugin through the 'Plugins' menu in WordPress
3. Go to Settings->Modal Read More and check the options
